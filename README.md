## 🇨🇳 中文介绍

**SoundSort** 是一款基于 Next.js 构建的网页应用，帮助你将网易云音乐歌单按音乐风格智能分类。

### ✨ 功能特点

- **导入歌单** — 粘贴网易云音乐歌单链接或 ID 即可导入
- **智能分类** — 基于歌曲名称、歌手、专辑等多维度信息，自动识别音乐风格
- **风格分布** — 可视化展示歌单中各音乐风格的比例
- **多标签分类** — 每首歌曲可拥有多个风格标签
- **搜索筛选** — 按风格分类浏览，按歌名/时长排序
- **深色主题** — 沉浸式音乐风格 UI 设计

## 🇬🇧 English

**SoundSort** is a Next.js web application that intelligently classifies NetEase Cloud Music playlists by music genre.

### ✨ Features

- **Playlist Import** — Paste a NetEase Cloud Music playlist link or ID to import
- **Smart Classification** — Multi-dimensional analysis based on song name, artist, album, and more
- **Genre Distribution** — Visual breakdown of genre composition
- **Multi-tag Classification** — Each song can have multiple genre tags
- **Search & Filter** — Browse by genre, sort by name or duration
- **Dark Theme** — Immersive music-inspired UI design

### 🎸 Supported Genres

Mandopop, Western Pop, Hip-Hop/Rap, R&B, Rock, Electronic, Folk, Jazz, Classical, Metal, Reggae, Country, Blues, Funk, Soul, Indie, K-Pop, J-Pop, ACG, GuFeng, Pop Rock, Alternative, Instrumental, Ambient, Latin, DnB, Breakbeat, House, Techno, Trance, Dubstep, Trap, Melodic Rap, Pop Rap, Lo-fi, Vaporwave, Dream Pop, Post-rock, Math Rock, New Wave, Punk, Hardcore, Emo, Shoegaze, Synthwave, City Pop, Disco, Jazz Rap, Soul Rap, Art Pop… and many more subgenres.

### 🚀 Quick Start

```bash
npm install
npm run dev        # Development
npm run build && npm start  # Production
```
